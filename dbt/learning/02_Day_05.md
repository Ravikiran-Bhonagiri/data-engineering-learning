# Day 05: Documentation & DAG

**Goal:** Document your dbt project.

---

## Add Descriptions

**File:** `models/marts/schema.yml`
```yaml
models:
  - name: dim_customers
    description: |
      Customer dimension table with lifetime metrics.
      Updated daily. Source: raw.customers + raw.orders.
    
    columns:
      - name: customer_id
        description: "Unique identifier for each customer"
      
      - name: lifetime_value
        description: "Total revenue from this customer (all time)"
```

---

## Generate & View Docs

```bash
dbt docs generate
dbt docs serve
```

Navigate to `localhost:8080`:
- Browse models
- See column descriptions
- View DAG (lineage graph)
- Search everything
