# Day 04: Testing

**Goal:** Implement data quality tests.

---

## Schema Tests (YAML)

**File:** `models/marts/schema.yml`
```yaml
models:
  - name: dim_customers
    description: "Customer dimension table"
    columns:
      - name: customer_id
        tests:
          - unique
          - not_null
      
      - name: email
        tests:
          - not_null
      
      - name: state
        tests:
          - accepted_values:
              values: ['NY', 'CA', 'TX', 'IL']
```

---

## Custom Data Test

**File:** `tests/no_negative_amounts.sql`
```sql
-- This should return 0 rows
-- If it returns rows, the test fails
SELECT *
FROM {{ ref('stg_orders') }}
WHERE amount < 0
```

---

## Run Tests

```bash
dbt test                       # All tests
dbt test --select dim_customers  # Specific model
```
